# Customer Churn Prediction
Author: Adebayo Aonullahi

Aim:

- To Predict if Customers will continue to subscribe to the service provide.
- To highlight major reasons why customers churn.
- To recommend possible solutions to curb the customer churn.


## Libraries

1. Pandas
2. Numpy
3. Matplotlib
4. Sklearn
5. Seaborn
